# How to Build
Better Video Player is available for download via the [Chrome Web Store](https://chromewebstore.google.com/detail/ekdiodfggfcfmhdmhehjemchfmocnbio)\
Alternatively, you can download the [latest GitHub tag](https://github.com/mrbooboostank/Better-Video-Player/tags) and load the "BetterVideoPlayer" folder as an unpacked extension\
The "Source" folder contains resources used to create Better Video Player's logo. The Source folder isn't required for using the extension

# Bug Reporting and Feature Requests
Bug reports and feature requests can be made via the [Issues tab](https://github.com/mrbooboostank/Better-Video-Player/issues) on this repository\
Please specify the affected Better Video Player version, your browser with its version, and all extensions you had enabled if you're reporting a bug
